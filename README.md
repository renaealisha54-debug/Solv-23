# Solv-23# Support Ticket System

A customer support platform built to streamline ticket tracking and resolution.

## Tech Stack
- **Framework:** React + TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **State Management:** React Query
- **Backend:** Supabase

## Getting Started
1. Install dependencies: `npm install`
2. Set up environment variables for Supabase (URL and ANON_KEY).
3. Run the development server: `npm run dev`

## Architecture
- All data fetching logic is located in `src/services/`.
- UI components follow the shadcn/ui library standards.
- Project-specific constraints and business logic definitions can be found in the AI Knowledge configuration.

## Contribution Guidelines
- Ensure all new components are documented in the `README` or added to the `components` directory.
- Follow the established naming conventions (kebab-case for files, PascalCase for components).
